# TMDB CLI ![TMDB](https://img.shields.io/badge/TMDB-01B4E4?style=for-the-badge&logo=themoviedb&logoColor=white)

A Command line tool to get the live movies data using TheMoviDatabase API

## Setup
1. Clone the repo
2. Create a '.env' file with you API key: TMDB_API_KEY =you_key_here
3. Install dependencies:
pip install requests pyhton-dotenv


## Usage
python main.py --popular
python main.py --upcoming
python main.py --now_playing
python main.py --top_rated

## Tech Stack

Python | REST API | Argparse | TheMovieDatabase API
